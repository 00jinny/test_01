# 🐍 Snake Game (HTML + CSS + JavaScript)

웹 브라우저에서 바로 실행할 수 있는 간단한 **스네이크 게임(Snake Game)** 프로젝트입니다.  
하나의 HTML 파일 안에 **HTML / CSS / JS**가 모두 포함되어 있어 별도 설치 없이 실행할 수 있습니다.

---

## 🎮 게임 특징

- 키보드 **화살표 키**로 조작  
- **R 키**로 게임 재시작  
- HTML5 `<canvas>` 기반 그래픽  
- 충돌 감지 및 점수 시스템 포함  
- 단일 HTML 파일로 구성되어 실행이 매우 간단함  

---

## 📂 파일 구성
snake.html


---

## ▶️ 실행 방법

1. 저장소를 클론하거나 `snake.html` 파일을 다운로드합니다.  
2. 파일을 더블 클릭하여 **브라우저에서 열기**만 하면 바로 플레이할 수 있습니다.

---

## ⌨️ 조작법

| 키 | 기능 |
|----|------|
| ↑ | 위로 이동 |
| ↓ | 아래로 이동 |
| ← | 왼쪽 이동 |
| → | 오른쪽 이동 |
| R | 게임 재시작 |

---

## 🛠️ 기술 스택

- **HTML5 Canvas**
- **CSS3**
- **JavaScript (Vanilla JS)**

---

## 📌 향후 개선 아이디어

- 난이도(속도) 조절 기능  
- 장애물 추가  
- 최고 점수 저장(localStorage)  
- 모바일 터치 조작 지원
